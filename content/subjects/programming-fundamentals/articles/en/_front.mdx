# Programming Fundamentals: Overview

This course introduces the core concepts of programming using Python and C++. You'll learn to think algorithmically, work with data structures, and write clean, efficient code.

## What You'll Learn

- Variables, types, and control flow
- Functions, modules, and error handling
- Arrays, lists, trees, and graphs
- Sorting and searching algorithms
- Time and space complexity analysis

## Prerequisites

No prior programming experience is required. A basic understanding of mathematics (algebra level) is helpful but not essential.
